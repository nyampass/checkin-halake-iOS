# checkin-halake-iOS

iOS app for [checkin-halake](https://github.com/nyampass/checkin-halake) api

# Setup

Add Settings.swift

```
struct Settings {
    static var APIBaseURL = "<API Server>/api"
    static var APIHeadersKey = "<API Key>"
}
```
